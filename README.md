# Детекция автомобильных номеров с использованием YOLOv11

Проект посвящен исследованию и практическому применению новейшей архитектуры **YOLOv11** для автоматического обнаружения государственных регистрационных знаков (ГРЗ) в видеопотоке. Работа выполнена в рамках учебной практики по разработке нейросетей.

## Основные возможности:
* **SOTA архитектура:** Использование YOLOv11 обеспечивает высокую точность и скорость обработки кадров.
* **Инференс на видео:** Обработка видеофайлов с автоматическим наложением Bounding Boxes на распознанные объекты.
* **Адаптация под железо:** Код поддерживает вычисления на GPU (NVIDIA CUDA / AMD ROCm), что критично для задач реального времени.

## Технологический стек:
* **Язык:** Python
* **Библиотеки ML:** `ultralytics` (YOLOv11), `opencv-python`
* **Инструментарий:** Jupyter Notebook, Google Colab, Gdown (для загрузки данных из облака)

## Краткое описание процесса:
1.  **Настройка среды:** Установка библиотек и проверка доступности GPU.
2.  **Загрузка данных:** Получение весов модели и тестовых видеоматериалов.
3.  **Детекция:** Запуск модели на видеопотоке с использованием библиотеки Ultralytics.
4.  **Результат:** Сохранение обработанного видео с визуализацией найденных номеров.
